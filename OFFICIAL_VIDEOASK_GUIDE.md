## VideoAskとは

**VideoAsk**は、Typeform社が提供する動画ベースのインタラクティブフォームプラットフォームです。

### 主な特徴
- 📹 **動画でのコミュニケーション**: 質問を動画で表示し、回答も動画・音声・テキストで収集
- 🎯 **高いエンゲージメント**: 従来のフォームと比較して回答率が大幅に向上（事例: 65% → 98%）
- 🔀 **条件分岐**: 回答に応じて異なる質問パスを作成可能
- 🔌 **豊富な連携**: Zapier経由で1,500以上のアプリと連携、API・Webhook対応
- 🎨 **9種類の回答タイプ**: テキスト、音声、動画、カレンダー予約、決済、NPSなど

### 主なユースケース
- **採用**: 非同期ビデオインタビュー
- **営業・マーケティング**: リード獲得、顧客フィードバック
- **カスタマーサポート**: FAQ、トレーニング、サポート

## 詳細な工程説明

### 【作成者側】工程1: アカウント作成・ログイン

**手順:**
1. **https://videoask.com** にアクセス
2. 「Sign Up」または「Get Started」をクリック
3. 以下のいずれかでサインアップ:
   - メールアドレス
   - Googleアカウント
   - Typeformアカウント
4. プラン選択（無料プランまたは有料プラン）

**プラン比較:**
| 機能 | 無料プラン | 有料プラン |
|------|----------|----------|
| ステップ数 | 制限あり | 無制限 |
| 録画時間 | 月間制限あり | より多い |
| 回答数 | 制限あり | プランによる |
| 分岐ロジック | ✓ | ✓ |
| カスタムブランディング | ✗ | ✓ |

**所要時間:** 2-3分

---

### 工程2: VideoAsk作成開始

**手順:**
1. ダッシュボードで「**Create your first videoask**」をクリック
2. 以下から選択:
   - **Start with a template** (テンプレートから開始)
   - **Start from scratch** (最初から作成)

**テンプレート例:**
- Customer Feedback（顧客フィードバック）
- Job Application（求人応募）
- Lead Qualification（リード選別）
- Product Demo（製品デモ）
- Event Registration（イベント登録）

**所要時間:** 30秒

### 工程5: 動画録画・アップロード（詳細版）

#### 5-1. 動画ソースの選択

以下のいずれかを選択:

**A. Webカメラで録画**
```
1. 「Record via webcam」をクリック
2. カメラ・マイクの許可を承認
3. デバイス選択（複数ある場合）
```

**B. 既存ファイルをアップロード**
```
1. 「Upload existing video」をクリック
2. ファイルを選択
3. 対応形式: MP4, MOV, AVI, WebM等
4. 最大ファイルサイズ: 500MB
```

**C. ストックライブラリから選択**
```
1. 「Choose from Pexels or Giphy」をクリック
2. キーワードで検索
3. 動画またはGIFを選択
```

**D. 画面録画**
```
1. 「Record a screen share」をクリック
2. 共有する画面/ウィンドウを選択
3. 録画開始
```

**E. 後で追加**
```
1. 「Add video later」を選択
2. プレースホルダーが表示される
3. 後からビルダーで追加可能
```

#### 8-2. 高度なロジック (Advanced Logic)

**高度なロジックとは:**
```
複数のif/then条件文を組み合わせて、
過去の回答を総合的に評価し、
次のステップを決定する機能
```

**有効化方法:**
```
Settings → Advanced Logic → Toggle ON
```

**条件の種類:**

| 条件タイプ | 説明 | 例 |
|-----------|------|-----|
| Answer to step | 特定ステップの回答 | Step 1 = "満足" |
| Score | スコア（NPS®など） | Score >= 8 |
| Multiple conditions | 複数条件の組み合わせ | Step 1 = "A" AND Score > 5 |

**優先順位:**
```
条件は上から下へ評価される
最初にマッチした条件が適用される

重要: 複雑な条件を上に配置！
```

**設定例:**
```
┌─────────────────────────────────────────┐
│  Advanced Logic Rules                   │
├─────────────────────────────────────────┤
│                                         │
│  Rule 1 (最優先):                       │
│  IF Step 1 = "非常に不満"               │
│  AND Step 2 contains "返金"             │
│  THEN → "緊急対応チーム"                │
│                                         │
│  Rule 2:                                │
│  IF NPS Score >= 9                      │
│  THEN → "推薦プログラム案内"            │
│                                         │
│  Rule 3:                                │
│  IF Step 1 = "満足"                     │
│  THEN → "次のステップ"                  │
│                                         │
│  Default (どれにも該当しない):          │
│  THEN → "終了画面"                      │
│                                         │
└─────────────────────────────────────────┘
```

**複雑なロジック例:**
```
【シナリオ: リード選別】

Step 1: 会社規模は？
├─ 1-10名 → Skip to "スモールビジネス向けプラン"
├─ 11-50名 → 次へ
├─ 51-200名 → 次へ
└─ 201名以上 → Skip to "エンタープライズ専用"

Step 2: 年間予算は？
├─ $1,000未満 → Skip to "セルフサービスプラン"
├─ $1,000-$10,000 → 次へ
└─ $10,000以上 → Skip to "プレミアムサポート"

Step 3: 導入時期は？
├─ 今すぐ → "営業チームに接続"
├─ 1-3ヶ月 → "ナーチャリングシーケンス"
└─ 3ヶ月以上 → "長期フォローアップ"

Advanced Logic:
IF 会社規模 = "201名以上" AND 予算 = "$10,000以上" AND 時期 = "今すぐ"
  → 直接アカウントマネージャーに通知 + ライブコール提案
```

**所要時間:** 3-10分（複雑さによる）

## コンタクトフォーム

### デフォルトフィールド

任意: *が表示されない
